## Overview

**Xiaomi Software Manager (XSM)** is a desktop application designed for **power users** who work with Xiaomi firmware and want a faster way to:

- Discover available firmware versions
- Compare releases across regions and variants
- Track updates over time
- Download official firmware files from original sources

The application provides automation for specific firmware discovery and retrieval tasks that are traditionally performed manually through a web browser, enabling more efficient workflows and clearer insight into available firmware releases, without acting as a replacement for Xiaomi services.

## Features

- Firmware discovery from publicly accessible Xiaomi pages
- Version comparison and tracking
- Manual, user-triggered downloads
- Multi-mirror fallback for improved reliability
- Desktop UI (no background services or daemons)
- No account login or authentication required

## Installation

Releases are published on the **GitHub Releases** [page](https://github.com/zilvmock/Xiaomi-Software-Manager/releases)

Download the `.zip` archive and extract it.

Run the portable `xsm.exe` executable.
